## 📚 Descripción del Proyecto

Esta actividad corresponde a la **Semana 2** del curso *Programadores por la Paz*, donde se aprenden los fundamentos de:

- 🖥️ **Terminal** - Interfaz de línea de comandos
- 📁 **Carpetas** - Organización de directorios
- 📄 **Archivos** - Creación y gestión de archivos
- 🔀 **Git** - Control de versiones
- ☁️ **GitHub** - Publicación y colaboración

## 📋 Contenido

### 1️⃣ Preguntas de Selección Múltiple
Respuestas a las 4 preguntas teóricas sobre el uso de la terminal.

### 2️⃣ Comandos de Terminal
Registro de todos los comandos utilizados durante la actividad práctica:
- `pwd` - Directorio actual
- `ls` - Listar archivos
- `mkdir` - Crear carpetas
- `cd` - Navegar directorios
- `touch` - Crear archivos

### 3️⃣ Reflexión
Análisis sobre la importancia de la terminal para los desarrolladores.

---

## 🛠️ Comandos Utilizados

| Comando | Descripción |
|---------|-------------|
| `pwd` | Muestra la ubicación actual |
| `ls` | Lista archivos y carpetas |
| `mkdir` | Crea una nueva carpeta |
| `cd` | Navega entre directorios |
| `touch` | Crea archivos vacíos |
| `git add` | Prepara archivos para commit |
| `git commit` | Guarda cambios localmente |
| `git push` | Sube cambios a GitHub |
